# ClipMate

ClipMate is a lightweight Microsoft Edge / Chrome extension for clipping web content to Notion.

This repository contains version-isolated snapshots:

- `clipmate-v0.1/` — v0.1 stable snapshot
- `clipmate-v0.2/` — v0.2 stable release candidate for Microsoft Edge Add-ons review

Current review version: `clipmate-v0.2`
