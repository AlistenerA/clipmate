# ClipMate

ClipMate is a lightweight Microsoft Edge / Chrome extension for clipping web content to Notion and Markdown.

## Current public release

This public repository contains the latest public release snapshot only:

- `clipmate-v0.3/` — current public release candidate

Older local development versions are not kept in this public repository. The local development repository may contain multiple version directories for development and rollback.

## Features

- Clip full articles or selected text from web pages
- Copy clean Markdown
- Save clips to Notion
- Markdown target profiles for Notion, Obsidian, Typora, and generic Markdown
- Improved handling for formulas, code blocks, images, links, and tables
- Local history and retry support
- Local-only settings and history storage

## Privacy

See [PRIVACY_POLICY.md](PRIVACY_POLICY.md).

ClipMate does not use analytics, ads, cloud sync, AI processing, or remote code execution. Notion data is sent directly from the browser to the official Notion API only when the user actively saves content.
