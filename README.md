# ClipMate

ClipMate is a lightweight Microsoft Edge / Chrome extension for clipping web content to Notion.

## Current public release

This public repository contains the latest public release snapshot only:

- `clipmate-v0.2/` — current public release

Older local development snapshots are not kept in this public repository to keep the repository small and easy to review.

## Release policy

The local development workspace may contain multiple version-isolated folders such as `clipmate-v0.1/`, `clipmate-v0.2/`, `clipmate-v0.3/`, and later versions.

This public GitHub repository only mirrors the latest version that is intended to remain public. Future versions may be open-source, partially open-source, or closed-source depending on the project stage. Closed-source or non-public parts will not be published here.
