## v0.4 功能摘要

ClipMate v0.4 **Site Profiles and Scenario Modes（站点适配与场景模式版）** 让插件更懂"当前页面是什么类型"，并根据页面类型使用更安全的剪藏策略。

### 7 大新增能力

| 功能 | 描述 | 状态 |
|------|------|:---:|
| **Page Type Detector** | 通用页面类型检测器，7 种类型（article/search/navigation/forum/video/ai-answer/unknown） | ✅ |
| **Site Profile Engine** | 19 个结构化 seed profiles + 纯函数匹配引擎 | ✅ |
| **Intent Signal Collector** | 14 种意图类型判断，脱敏 IntentSnapshot | ✅ |
| **Navigation Summary** | 导航/搜索页生成安全摘要，selection-first 优先 | ✅ |
| **Comment / Selection Clip** | 7 种选区上下文识别，不自动抓取评论 | ✅ |
| **Site Visual** | favicon/themeColor 安全提取 + 纯函数 cache strategy | ✅ |
| **Link Card Preview** | linkCard builder + Markdown serializer，4 种 source | ✅ |

## 从 v0.3 到 v0.4 的主要变化

### 新增
- Page Type Detector：从 4 类扩展到 7 类，通用启发式
- Site Profile Engine：结构化站点规则，19 个 seed profiles
- Intent Signal Collector：综合 pageType + siteProfile + selection + visible 信号
- Navigation Summary Mode：导航/搜索页安全摘要替代全文噪音提取
- Comment / Selection Clip Mode：7 种选区模式
- Site Visual Metadata：favicon/themeColor 安全提取
- Link Card Preview Core：链接卡片构建与 Markdown 序列化

### 改进
- Article Boundary Guard：IS01 修复，低置信+高链接密度路径可触发
- metaParser：icon 提取委托给 siteVisual safe extractor（M2 DRY cleanup）
- 代码质量：4 次 Anti-Slop Review 全部通过

### 不变
- manifest 权限未变更（storage/activeTab/api.notion.com）
- 依赖未增加（Readability/React/Turndown）
- Popup/Options/Background/Notion 保存主链路未变
- selection-first 行为不变

## 安全与隐私边界

### 不持久化的数据
- IntentSnapshot：仅在 tab 内存，不写 storage
- 完整 DOM/HTML：不保存
- 评论全文：不自动抓取，仅保存用户选区
- 正文全文：不在 LinkCard/NavSummary/CommentDraft 中保存

### 不访问网络的功能
- Site Visual：图标/主题色仅读取当前 Document DOM
- Link Card：不访问目标 URL
- Navigation Summary：链接仅提取 DOM `a[href]`，不抓取目标内容
- Page Type Detector / Site Profile Engine / Intent Signal Collector：均为纯函数或只读 DOM

### 不新增
- manifest 权限
- npm 依赖
- AI/OCR/云同步/付费功能
- 远程代码

## 已知限制

- 仅支持 Notion，不支持飞书/语雀
- 手动 Token 配置（非 OAuth）
- 无 AI 摘要/标签/OCR
- Seed profile selector hints 为 seed 级别，部分站点需真实浏览器手动验证
- 短视频/AI 对话页 DOM 高度动态，selector hints 可能不匹配
- 搜索页动态加载（infinite scroll）首次快照可能不完整
- Site Visual 不验证 favicon 可访问性（不发起网络请求）
