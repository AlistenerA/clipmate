## 一、Vibe Coding 核心事实

- **Andrej Karpathy（OpenAI联合创始人）** 2025年2月提出概念，同年被《柯林斯词典》评为年度词汇
- 当前 **YC 25%的初创公司代码由AI生成**，"想法→产品"门槛已崩塌
- 核心机会：**代码不再是护城河，分发渠道和行业认知才是**

## 三、推荐入门实操方案（30天计划）

### 👉 推荐路径：Chrome插件起步 → 验证 → 扩展到SaaS

#### 第一阶段：准备（Day 1-3）
```
工具准备：
□ 注册 Cursor Pro（$20/月）或 字节跳动Trae（免费，国产替代）
□ 注册 Supabase（数据库，免费额度足够起步）
□ 注册 Vercel（前端部署，免费）
□ 注册 Stripe（国外收款）或 PayJS（国内收款）
□ 准备一个域名（$10-15/年）

成本总计：约 $50/月（¥350/月）
```

#### 第二阶段：选题验证（Day 4-7）
```
执行步骤：
1. 打开 Chrome应用商店，搜索3个你感兴趣的词（如 "writing"、"productivity"、"screenshot"）
2. 找出排名前10中评分低于3.5星的插件
3. 读差评，总结用户真正的痛点
4. 挑一个痛点最明确、用户量最大的
5. 验证：在Twitter/Reddit/知乎发帖问"你希望XX工具有什么功能？"
```

**当前高需求选题参考：**
- AI驱动的PDF页面内容一键翻译
- 网页截图+自动打码敏感信息
- LinkedIn好友备注+标签智能管理
- 网页内容一键整理为Notion/飞书文档

#### 第三阶段：开发（Day 8-14）
```
用 Cursor/Claude 对话式开发：

Prompt 模板：
"帮我用 Next.js + TypeScript 创建一个Chrome插件，
功能是 [你的功能描述]，
需要包含：
1. 点击图标弹出浮窗
2. 调用 [API名称] API
3. 结果展示和复制按钮
4. 设置页面（API Key配置）
技术栈：React + Tailwind CSS + Plasmo框架"
```

#### 第四阶段：上线发布（Day 15-21）
```
发布清单：
□ Chrome应用商店提交审核（一次性费用 $5）
□ 制作一个30秒产品演示GIF
□ 写一篇 ProductHunt 风格的发布文案
□ Twitter/即刻/知乎发布产品介绍
□ 在你的目标用户社区发帖（如 V2EX、Reddit对应sub）
```

#### 第五阶段：增长变现（Day 22-30）
```
变现节奏：
□ Day 1-14：免费，收集用户反馈
□ Day 15+：增加 Pro 版本（$5-9/月），免费版加限制
□ Day 30+：看数据决定是否扩展为独立SaaS

定价参考：
- Chrome插件：$5-15/月 或 $19-49 一次性
- 付费转化率通常 2-5%（免费→付费）
- 目标：1000个免费用户 → 20-50个付费 → $100-750 MRR
```

## 五、关键避坑指南

1. **不要All-in一个产品**：多个小产品＞一个大产品（IndieHackers数据验证）
2. **先有用户再有付费**：永远先验证需求，再写代码
3. **代码不是护城河**：分发渠道（Twitter/SEO/社群）才是核心竞争力
4. **别在技术选型上纠结**：Next.js + Supabase + Stripe 解决90%场景
5. **API Key别泄露**：Vibe Coding很容易把密钥写进代码，务必用环境变量
6. **别做纯AI聊天工具**：竞争太大，做垂直场景的AI工具
7. **国内注意合规**：小程序需要企业资质，建议先用Chrome插件/独立站起步
