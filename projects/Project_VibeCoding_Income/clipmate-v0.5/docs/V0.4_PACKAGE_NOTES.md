## package.json 变更

### version

`0.3.0` → `0.4.0`

### zip script

原脚本：

```json
"zip": "powershell -Command \"Add-Type -AssemblyName System.IO.Compression.FileSystem; $zip='clipmate-v0.3.zip'; if(Test-Path $zip){Remove-Item $zip}; [System.IO.Compression.ZipFile]::CreateFromDirectory('dist', $zip)\""
```

变更为：

```json
"zip": "powershell -Command \"Add-Type -AssemblyName System.IO.Compression.FileSystem; $zip='clipmate-v0.4.zip'; if(Test-Path $zip){Remove-Item $zip}; [System.IO.Compression.ZipFile]::CreateFromDirectory('dist', $zip)\""
```

唯一变更：输出文件名从 `clipmate-v0.3.zip` 改为 `clipmate-v0.4.zip`，与版本号同步。打包逻辑 (`Add-Type` / `CreateFromDirectory`) 未改变。无新增依赖。

## 构建流程

```
npm run lint   → eslint src --ext .ts,.tsx
npm run test   → vitest run
npm run build  → tsc --noEmit && vite build
npm run zip    → powershell [System.IO.Compression.ZipFile]::CreateFromDirectory('dist', $zip)
```
